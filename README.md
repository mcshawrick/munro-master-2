# Munro Master

The ultimate Munro bagging app - track all 284 Scottish mountains with GPS, weather, and gamification.

Gotta Bag 'Em All!

## Features
- All 284 Munros with photos
- GPS tracking and nearby alerts  
- Weather integration
- User profiles and achievements
- Mobile-first PWA

## How to Use
Open the live site and add to your home screen for the app experience.
